# es6-boilerplate

## Overview

gulp template.

## Description

- gulp
- jade
- stylus(sanitize & FLOCSS)
- ES2015
- browserify(babel)
- image-min
- browser reload

## Requirement

- node.js (npm)

## Usage

  % npm install
  % gulp run

## Command

  % gulp build // compile
  % gulp run // watch and compile
