# es6-boilerplate
![image](http://sadakoa.minibird.jp/img/boi.png)

## Overview

gulp template.

## Description

- gulp
- jade
- stylus(sanitize & FLOCSS)
- ES2015
- browserify(babel)
- image-min
- browser reload

## Requirement

- node.js (npm)

## Usage

  % npm install

  % gulp

## Command

  % gulp build // compile

  % gulp // watch and compile
